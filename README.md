# 海外华人国际机票引流网站

一个面向海外华人的国际机票代理网站，部署在Vercel，全球CDN加速。

## 功能特点

- **特价机票**: 实时更新的全球航线特价信息
- **留学生机票**: 专属优惠，额外行李额
- **团队机票**: 3人成团，专属折扣
- **改签退票**: 7×24小时售后响应
- **行程规划**: 一对一出行方案
- **出行攻略**: 转机、行李、入境政策指南

## 技术栈

- Next.js 14 (App Router)
- TypeScript
- Tailwind CSS
- Vercel 部署

## 开发

```bash
# 安装依赖
npm install

# 本地开发
npm run dev

# 构建
npm run build
```

## 部署

### 方法1: Vercel CLI

```bash
npm i -g vercel
vercel
```

### 方法2: GitHub 集成

1. 将代码推送到 GitHub 仓库
2. 访问 [Vercel](https://vercel.com) 并使用 GitHub 登录
3. 点击 "New Project" 并导入仓库
4. 配置完成后自动部署
