# SkyPath Global - 国际机票代理平台

## 1. Project Overview

**项目名称**: SkyPath Global (天航国际)
**类型**: B2C/B2B 国际机票服务平台
**核心功能**: 航班比价 | 航司选座值机 | 签证查询 | 企业服务 | 人工客服
**目标用户**: 个人旅客 + 企业差旅客户

---

## 2. UI/UX Specification

### Layout Structure

**页面结构 (单页应用)**:
- 固定顶部导航栏 (80px)
- Hero 搜索区域 (100vh - 80px)
- 航班比价展示区
- 航司服务区 (选座/值机)
- 签证查询区
- 企业服务区
- 人工客服区
- 更多功能展示区
- 页脚

**响应式断点**:
- Desktop: ≥1200px
- Tablet: 768px - 1199px
- Mobile: <768px

### Visual Design

**色彩系统** (高端商务风格):
```css
--primary: #0A1628;        /* 深夜蓝 - 主背景 */
--primary-light: #132238;  /* 浅深夜蓝 - 卡片背景 */
--accent: #00D4AA;         /* 翡翠绿 - 强调色 */
--accent-hover: #00F5C4;   /* 亮翡翠 - 悬停态 */
--gold: #C9A962;           /* 香槟金 - VIP/企业 */
--gold-light: #E8D5A3;     /* 浅金 */
--text-primary: #FFFFFF;   /* 主文字 */
--text-secondary: #8B9CB3; /* 次要文字 */
--text-muted: #5A6A7E;     /* 弱化文字 */
--border: #1E3A5F;         /* 边框色 */
--success: #22C55E;        /* 成功 */
--warning: #F59E0B;        /* 警告 */
--danger: #EF4444;         /* 错误/低价 */
```

**字体系统**:
- 标题: "Playfair Display", serif (优雅高端)
- 正文: "DM Sans", sans-serif (现代易读)
- 数字/价格: "JetBrains Mono", monospace (清晰数字)

**字号**:
- H1: 56px / 64px (hero标题)
- H2: 40px / 48px (区块标题)
- H3: 24px / 32px (卡片标题)
- Body: 16px / 24px
- Small: 14px / 20px

**间距系统** (8px基础):
- xs: 4px
- sm: 8px
- md: 16px
- lg: 24px
- xl: 32px
- 2xl: 48px
- 3xl: 64px
- 4xl: 96px

**视觉效果**:
- 卡片: 毛玻璃效果 (backdrop-filter: blur(20px))
- 阴影: 0 8px 32px rgba(0, 0, 0, 0.3)
- 边框: 1px solid rgba(255, 255, 255, 0.08)
- 渐变: 微妙的径向渐变背景
- 动画: 页面加载时交错渐入

### Components

**导航栏**:
- Logo (左侧)
- 主导航链接 (居中): 航班搜索 | 航班比价 | 选座值机 | 签证查询 | 企业服务
- 右侧: 登录/注册按钮, 客服热线

**搜索组件**:
- 出发地/目的地输入 (带自动完成)
- 日期选择器 (去程/返程)
- 乘客数/舱位选择
- 搜索按钮 (渐变背景)

**航班卡片**:
- 航司Logo + 名称
- 出发/到达时间 + 机场
- 航程时长 + 经停信息
- 价格展示 (原价/现价)
- 立即预订按钮

**航司服务卡片**:
- 航司Logo网格
- 选座/值机/行李/餐食等服务图标
- 快捷入口按钮

**签证查询卡片**:
- 国家名称 + 国旗
- 签证类型
- 办理时长 + 费用
- 立即咨询按钮

**企业服务卡片**:
- 企业名称/联系人输入
- 需求类型选择 (企业差旅 | 团票 | 定制)
- 提交按钮

**客服组件**:
- 浮动客服按钮 (右下角)
- 在线状态指示器
- 点击展开聊天窗口

## 3. Functionality Specification

### 核心功能

**1. 航班搜索与比价**
- 多城市搜索 (单程/往返/多程)
- 智能推荐低价航班
- 价格日历展示
- 筛选: 航司/起降时间/中转/舱位

**2. 航司服务**
- 全球50+主流航司Logo展示
- 选座服务入口 (各航司官网跳转)
- 在线值机入口
- 行李额度查询
- 特殊餐食预订

**3. 签证查询**
- 200+国家/地区签证信息
- 签证类型/材料/时长/费用
- 可更新的签证规则库
- 在线咨询服务

**4. 企业服务**
- 企业账号注册
- 差旅管理方案
- 团票预订
- 月结付款
- 专属客服

**5. 人工客服**
- 7x24小时在线客服
- 电话热线显示
- 微信/WhatsApp联系方式
- 常见问题FAQ

**6. 会员系统**
- 个人用户注册/登录
- 订单管理
- 收藏航班
- 会员等级 (普通/银卡/金卡/黑卡)

### 用户交互

- 搜索表单即时验证
- 价格数字滚动动画
- 卡片悬停放大效果
- 页面滚动触发动画
- 客服弹窗平滑展开
- 表单提交成功提示
